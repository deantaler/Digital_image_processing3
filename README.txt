To run the program you should insert the following arguments:
1. right image
2. left image - should has 60% overlap to right image
3. n - best feature points
4. m - the best corresponding feature points for a selected feature point

after running the program you should select and click on a point on the left image,
the program will open a output image with matching between that point to m points of right image
in addition the program will write all those points to a text file named 'display.txt'

attached 2 images img_L, img_R and output image for selecting a specific point and display.txt
